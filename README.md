📚 Heritage Library Catalog (JavaScript Data Processing Project)

This project builds a complete data processing system for a fictional library catalog using JavaScript. It demonstrates how raw text data can be cleaned, structured, validated, grouped, filtered, and exported into multiple formats.

🚀 Features
Parses raw catalog strings into structured JavaScript objects
Cleans and normalizes data (trim, fallback values, type handling)
Validates entries for missing or invalid fields
Filters books by author using case-insensitive search
Groups books by decade (e.g., 1970s, 1980s)
Handles edge cases like unknown or missing data
Formats entries into readable library cards
Exports data to:
JSON format
CSV format
Calculates summary statistics:
Total number of books
Number of decade groups
Oldest and newest publication years
🧠 Key Concepts Used
Arrays and objects
Loops (for loops)
String manipulation (split, trim, toLowerCase, includes)
Conditional logic
Data validation
Object grouping and aggregation
JSON and CSV export formatting
📊 Example Output Features
Book search by author (e.g. “king”, “asimov”)
Decade-based grouping (e.g. 1950s, 1980s)
Clean formatted library card display
Export-ready JSON and CSV strings
🎯 Purpose

This project simulates real-world data processing tasks like those used in library systems, search engines, and backend data pipelines. It focuses on transforming messy raw data into structured, usable formats.
